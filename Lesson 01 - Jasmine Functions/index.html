Jasmine Functions

Calculator

In this lesson I learnt about the various functions available to Jasmine. I learnt how to configure suites, specs and expectations. To perform some tests, I created a calculator that will be used to define specs for testing a calculator class.

Calculator class

I first started by creating a calculator class with various methods that we would use to subtract, add, multiply and divide numbers passed to the class.

  1. class Calculator {
  2.     constructor() {
  3.         this.total = 0;
  4.     }
  5.  
  6.     add(number) {
  7.         return this.total += number;
  8.     }
  9.  
  10.     subtract(number) {
  11.         return this.total -= number;
  12.     }
  13.  
  14.     multiply(number) {
  15.         return this.total *= number;
  16.     }
  17.  
  18.     divide(number) {
  19.         if(number === 0) { 
  20.             throw new Error('Cannot divide by zero'); 
  21.         }
  22.  
  23.         return this.total /= number;
  24.     }
  25. }
Calculator UI

Here I created a form input that execute some calculations on submission. With the use of a regular expression I was able to match the operator used and perform the desired calculation passed in the form input. An event listener attached to the form input is used to pass the value of the input to a function that performs validations and carries out the calculations.

  1. const calculate = (input_value) => {
  2.     /**
  3.      * Below regural expression(regex) is for matching our operators (+, -, * and /).
  4.      * In JavaScript we wrap regexes with a slash in the beginning and end of the expression.
  5.      * This regex was generated here: https://regex101.com/
  6.     */
  7.     const expression = /\+|\-|\*|\//;
  8.  
  9.     // Here we split the string from the input into an array
  10.     const numbers = input_value.split(expression);
  11.  
  12.     /**
  13.      * Get the first and second number in the arithmetic operation.
  14.      * Because they are strings we use parseInt() to convert them to numbers.
  15.     */
  16.     const first = parseInt(numbers[0]);
  17.     const second = parseInt(numbers[1]);
  18.  
  19.     /**
  20.      * The match helper is a string prototype that retrieves the result of matching a string against a
  21.      * regex. The result of it is returned as an array. In this case it will match the operator and return it.
  22.      * 
  23.      * More info: https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/match
  24.     */
  25.     const operator_match = input_value.match(expression);
  26.     let operator;
  27.  
  28.     if(Number.isNaN(first) || Number.isNaN(second) || operator_match === null) {
  29.         return updateResult('Operation not recognized');
  30.     } else {
  31.         // get the operator from the match result array
  32.         operator = operator_match[0];
  33.     }
  34.  
  35.     const calculator = new Calculator();
  36.     calculator.add(first);
  37.  
  38.     let result;
  39.  
  40.     switch (operator) {
  41.         case '+':
  42.             result = calculator.add(second);
  43.             break;
  44.         case '-':
  45.             result = calculator.subtract(second);
  46.             break;
  47.         case '*':
  48.             result = calculator.multiply(second);
  49.             break;
  50.         case '/':
  51.             result = calculator.divide(second);
  52.             break;
  53.         default:
  54.             break;
  55.     }
  56.  
  57.     return updateResult(result);
  58. };
  59.  
  60. const updateResult = (calculator_result) => {
  61.     const calculator_result_element = document.getElementById('calculator-result-span');
  62.  
  63.     if (calculator_result_element) {
  64.         calculator_result_element.innerText = `Result: ${calculator_result}`;
  65.     }
  66. };
  67.  
  68. document.getElementById('calculator-input').addEventListener('keyup', (event) => {
  69.     if (event.key === 'Enter') {
  70.         return calculate(event.target.value);
  71.     }
  72. });

Calculator Suite

Here, I was introduced to the concepts of jasmine and it's functions. I got to learn how suites are defined by the describe keyword and the structure of specs within it. I also got to see how to use the spec html file provided by jasmine to see the results of the defined specs.

  1. /**
  2.  * A suite in jasmine is a group of specs that we define to
  3.  * test a give block of JavaScript code.
  4.  * 
  5.  * To disable a suite, we use the xdescribe() function which inturn 
  6.  * disables all the specs under it.
  7. */
  8. describe('calculator.js', () => {
  9.     /**
  10.      * Specs are a set of expectations that test a block
  11.      * of code. Each spec has a title and callback function that
  12.      * houses the expectations.
  13.     */
  14.     it('should add numbers to total', () => {
  15.         /**
  16.          * An expectation is an assertion that tests a statement 
  17.          * returning a boolean value of the result.
  18.         */
  19.         const calculator = new Calculator();
  20.         calculator.total = 40;
  21.         calculator.add(5);
  22.  
  23.         expect(calculator.total).toBe(45);
  24.     });
  25.  
  26.     /**
  27.      * When we want to disable a spec, we use the xit() function.
  28.      * This temporarily disables a spec and marks it as pending
  29.      * which makes it not to be executed.
  30.     */
  31.     xit('should subtract numbers from total', () => {
  32.         // Expectations go here
  33.         const calculator = new Calculator();
  34.         calculator.total = 40;
  35.         calculator.subtract(5);
  36.  
  37.         expect(calculator.total).toBe(35);
  38.     });
  39.  
  40.     it('should multipy total by number', () => {
  41.         // Expectations go here
  42.         const calculator = new Calculator();
  43.         calculator.total = 40;
  44.         calculator.multiply(2);
  45.  
  46.         expect(calculator.total).toBe(80);
  47.     });
  48.  
  49.     it('should divide total by number', () => {
  50.         // Expectations go here
  51.         const calculator = new Calculator();
  52.         calculator.total = 40;
  53.         calculator.divide(10);
  54.  
  55.         expect(calculator.total).toBe(4);
  56.     });
  57. });
